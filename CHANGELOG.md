# Changelog

All notable changes to this project will be documented in this file.

---

## v0.4.1 - Know Yourself

### Added

- 新增 `recovery-plan.md` 中的 **计划书单** 模块。
- 新增 `body-profile.md` 中 **身体不适-偶发** 模块。

## v0.4.0 - Know Yourself

### Added

- 将项目名称正式确定为 **Know Yourself**。
    
- 明确项目定位：认识自己，重新生活。
    
- 新增 `body-profile.md` 中的 **Self Awareness（自我认知）** 模块。
    
- 新增 **Energy Map**，记录充电与耗电事件。
    

### Changed

- 项目目标从「身体康复」扩展为「建立适合自己的生活方式」。
    
- 将恢复思路调整为三个阶段：
    
    - 恢复生活
        
    - 恢复运动
        
    - 恢复健康
        
- 调整 README，重新整理项目结构与理念。
    

---

## v0.3.0 - Recovery Framework

### Added

- 新增 `recovery-plan.md`。
    
- 建立三阶段恢复框架。
    
- 建立长期记录（docs）与每日记录（journal）的结构。
    

### Changed

- 不再制定严格的每日运动计划。
    
- 改为根据身体状态选择当天运动。
    
- 强调「每天舒服地运动」，而不是完成训练任务。
    

---

## v0.2.0 - Journal

### Added

- 新增 `journal/`。
    
- 建立每日记录习惯。
    
- 新增 Tags 分类。
    
- 增加身体、饮食、运动、情绪等观察内容。
    

### Changed

- Journal 从固定模板改为自由记录。
    
- 鼓励记录当天真实发生的事情，而不是完成任务。
    

---

### Added

- 创建项目。
